ReactiveCanvas

ReactiveCanvas is a creative art project lauched since December 2020. It is an interactive system that allows everyone to generate own artwork with customizable inputs and rulesets. People can enjoy interacting with their generated arts and download. As a new kind of art form, ReactiveCanvas tries to involve computation and hands-on experience in the generating process of artworks.

Three.js is a cross-browser JavaScript library and application programming interface used to create and display animated 3D computer graphics in a web browser using WebGL.

ReactiveCanvas is computed with the celular autometa algorithm based on the Three.js framework. The art of ReactiveCanvas is generated in dynamic 3D-dimension. The celular autometa algorithm encodes the diversity of color, form, space, and speed. It encourages people to create art freely and interact with it from different perspectives.

The RhinoScriptSyntax module contains hundreds of easy-to-use functions that perform a variety of operations on Rhino, a 3D surface modeler software.

With the diversity of color, form, and space, there are tons of potentials of using the output from ReactiveCanvas for the experiment of spatial artwork. The simplified snippet code in python (Rhino Script Syntax) will be shared in the future for 3D creative enthusiasm.
